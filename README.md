Propozycja projektu:
Aplikacja wielojęzyczna Django i nextJS , do wyświetlania i zapisywania historii pogody
strona dla niezalogowanego użytkownika - wyświetla pogodę z wybranej lokalizacji lub lokalizacji użytkownika (poprawny dostęp do lokalizacji użytkownika, prośba o dostęp itp.)
zalogowany użytkownik - może zapisać swoje ulubione lokalizacje, może zapisać x (określone przez admina) prognoz.
admin - może wyświetlać i wyszukiwać zapisane prognozy użytkowników, zobaczyć statystyki miejsc o które pytali zalogowani i niezalogowani użytkownicy
dane pogodowe z zewnętrznego serwisu (np. OpenWeatherApp)
dane zapisywane w bazie danych 
wszystko oparte na docker compose (jedno polecenie u pobrania, ustawienia i zainstalowania lokalnie wszystkich elementów aplikacji)
